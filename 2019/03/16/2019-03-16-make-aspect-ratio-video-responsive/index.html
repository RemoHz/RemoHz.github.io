Make Aspect Ratio Video Responsive

Posted on 2019-03-16 | Visitors: |

To add an embedded video in HTML is as simple as add an image, but to make video responsive is a little bit tricky. When I try to make a full screen layout with video like below, it took my some effort to make it responsive. As my client request, this layout has below features:

  • Support both raw HTML5 <video> and <iframe> video (Vimeo), which they can easily modify in CMS.
  • All videos will be in a common aspect ratio 16/9.
  • Keep video height to fit the screen height in large screen, while keep video respect ratio in small screen.


Raw HTML5 Video Responsive

For raw video element, it’s not that hard and we can treat it as image element. We just need to set video’s max height, then the video size will be adjust based on browser window change.

Video works nicely as you scale down the browser window. But how can we locate the left and right element? You may say just make video’s parent element relative, then make left & right element absolute and negative to the video.

That’s what I thought initially. But the reality is that video element width is not what we can see(visible area), it actually takes the full width of window. So the previous solution will make the left & right element out of the browser window.


A better solution for this is to calculate the gap between video and window by using aspect ratio. Then we can locate the left & right element based on the gap correspondingly.

Iframe Video Responsive

For iframe video, if you only want to make video responsive and don’t need to take care of my specific layout case, then I suggest you to use FitVids.js, which can fix the issue quickly. Basically what it does is mainly wrap the iframe in a container and add the aspect ratio treatment, like below:
