PHP Crawler

PHP Crawler

Due to my current project, I need to get all units data from Monash University website both by code and faculty.

As above link, there are not API or web service provided to fetch those data and we must collect those data manually?

This is qutie common in data collection because the organization do not have the responsibility and duty to create a API for you a specific user.

So what we need to do is to collect those data by “hand”.

The hand here is what we called web crawler.

There are heaps of script languages and libs support crawler, but as a PHPer I prefer to use php realize it.

After take some time to research , I found the Phpfetcher is a qutie simple and easy-to-handled framework. It covered our most requirements like HTTP header and depth search.

To take the Monash website as example, the html code of unit is as below:

<li><a class="fixed" href="ACB1000.html">ACB1000</a> Principles of accounting and finance</li>

So to get all <li> tags is our mission. Please take care about each of tag here is an object, not array.

class mycrawler extends Phpfetcher_Crawler_Default {
    public function handlePage($page) {
        $fp = fopen('unitsByCode.csv', 'a');
        // Get All li
        $res = $page->sel('//li');
        for ($i = 0; $i < count($res); ++$i)
        {
            $name = "";
            $name = $res[$i]->plaintext;

            if ($name == "")
            continue;
            // Get <a>
            $objLi = $res[$i]->find("a");

            for ($j = 0; $j < count($objLi); ++$j)
            $url = $objLi[$j]->getAttribute('href');

            $compare = substr_compare($name, $url, 0, 7);

            if ($compare == 0)
            {
                echo $name."\n";
                echo "http://www.monash.edu.au/pubs/2016handbooks/units/".$url."\n\n";
                $unit = array($name, 'http://www.monash.edu.au/pubs/2016handbooks/units/'.$url);
                fputcsv($fp, $unit);
            }
        }
        fclose($fp);
    }
}
