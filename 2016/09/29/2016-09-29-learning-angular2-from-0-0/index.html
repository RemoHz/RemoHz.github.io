Learning Angular2 From Zero (Entry00)

Posted on 2016-09-29

Learning Angular2 from Zero (00) Start

15th Sep 2016, Google annuounced the final release version of Angular2. This is the most stable version which means the efficiency and performance have made a huge progress.

As a entry level front-end developer, it’s quite lucky that I could pick Angular2 as my entry framework. Due to the big difference between Angular1 and Angular2, I’m at the same start line comparing to other experienced Angular1 developer, which makes me feel a little bit confident and easy :stuck_out_tongue_winking_eye:. And also I really recommend Angular2 to the entry level front-end users just like me. Only if you have some basic object-oriented programming experience, you could master the Angular2 quickly. Most of the Typescript syntax is just like writing Java programs and you will enjoy to change back-end development into front-end development easily.

So from this post, I wanna record my learning journey for the Angular2. Most of relaed materials are from Udemy Angular2 course and Angular2 official page, which could find the link in Reference.

2. Environment Setup

Follow the below order to set up the development environment.

2.1 Install Node.js

2.2 Install NPM

$ npm install -g typescript // May need ‘sudo’ to get superuser permission

$ npm install -g typings // Again, may need ‘sudo’ to get superuser permission

|– app
| |– app.component.js
| |– app.component.js.map
| |– app.component.ts // Root component of application
| |– boot.ts // Starting modules of application
| |– boot.js.map
| -- boot.js |-- package.json // Standard Node package configuration |-- tsconfig.json // Configuration for Typescript compiler (Typescripty => Javascripty) |-- typings.json // Typescript definition configuration– index.html

$ cd ~/Angualr2-seed/ // The path to your Angular2 seed project directory

$ npm install

|– app
| |– app.component.js
| |– app.component.js.map
| |– app.component.ts // Root component of application
| |– boot.ts // Starting modular of application
| |– boot.js.map
| -- boot.js |-- package.json // Standard Node package configuration |-- tsconfig.json // Configuration for Typescript compiler (Typescripty => Javascripty) |-- typings.json // Typescript definition configuration |-- index.html |-- typings– node_modules // Installed dependencies

$ npm start

```
This will automatically compile the typescripte code into javascript and run it in you browser.
