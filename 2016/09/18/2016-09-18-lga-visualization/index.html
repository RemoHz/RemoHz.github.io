Melbourne LGA Profile Visualization

Melbourne local government areas profile Visualization

With the rapid rise in house prices, more and more citizens tend to invest the real estate. So there is a need for them to visualize which ares are worth to invest more and can get more return in the future. Therefore I plan to create an app to present the data visualization and provide references for them to consider. The tools I used is Leaflet and Carto.

Generally, there are two steps. One is data collection and clean. Another is data visualization.

After searched in Victorian government data directory, I got the raw data here which the type is Leaflet default geojson. There are almost 140 attributes inside, which it’s too much for us. So we need to remove the extra column of it. Here we used is the Carto which it’s a super hero to handle with geojson file. Below is the final dataset we need.

Than, what we need is to customise the Leaflet and import the dataset. Below is the code pieces. You can got the souce code in Github.
