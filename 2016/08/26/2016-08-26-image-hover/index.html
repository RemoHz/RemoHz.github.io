Display image on link hover(CSS)

Display image on link hover

Here is what we want displayed below. When the mouse hover on the QR icon, the QR image will display left beside it for readers to scan.

Actually there two ways to realize it.

One is using javascript to bind events both in onmouseover and onmouseout of the icon element to detect the mouse change.

Another method is using pure CSS. Here is the code:

a:hover {
    position: relative;
}
a:hover:after {
    content: url(url)            /* Image URL */
    display: block;
    position: absolute;
    left: 50px;                   /* change this value to one that suits you */
    top: 0px;                     /* change this value to one that suits you */
